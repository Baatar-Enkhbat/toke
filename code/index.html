Location
アクセス

駅近でフラットなルート。
都心へ便利なアクセス。

千葉市緑区土気町にある土気駅は、JR外房線の駅です。当駅は市内の
東側に位置して、日中は1時間あたり4本、朝の上り方面面と夕方の下り方面は
さらに運行本数が増えます。主な駅のアクセスとして、
千葉駅・京成千葉駅までは約21分、西船橋へも近いです。
